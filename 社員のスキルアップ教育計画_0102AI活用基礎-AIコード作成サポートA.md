# AIコード作成サポート(VSCode活用 基礎編)
AIコード作成とは、AIを活用してプログラムの作成や修正を効率化する手法です。

従来はすべて人がコードを書く必要がありましたが、現在はAIが以下のような支援を行います：

* コードの自動補完
* エラーの修正提案
* 設計や処理の提案

本資料では、AIコード作成の基本的な使い方と、ツールの選び方を解説します。

👉 **まずは全体像を理解し、自分に合った使い方を選べるようになることを目的とします**

---

## 1. はじめに：2つの使い方があります

AIコード作成には、大きく2つの使い方があります。

* **補助型( シンプル )**：コードの続きを提案してもらう
* **提案型( エージェント )**：やるべき作業ごとAIに任せる

初心者の方は、まずこの違いを理解してください。

---

## 1.1 最初に選ぶべき使い方

本資料では、以下の2つの始め方を用意しています。

---

### ■ パターンA：安全に学ぶ( 推奨 )

* 無料で使える補助型ツールを利用
* 軽量なモデルを使用

#### 理由

* 無料で始められる
* 動作がシンプルで理解しやすい
* 「自分で書く + AI補助」という形になるため学習効果が高い

👉 **まずはこちらで「AIに慣れる」ことを優先してください**

---

### ■ パターンB：効率重視で開発する

* 提案型( エージェント )ツールを利用
* 無料枠のあるモデルを使用

#### 理由

* AIが作業手順を提案してくれる
* 複数ファイルの修正などを自動で行える
* チャット型よりも無駄なやり取りが減り、無料枠でも効率よく使える

👉 **「とにかく早く作りたい人」向けです**

---

## 1.2 なぜ無料構成から始めるのか

AIは使い方によってコストが大きく変わります。

特にチャット型の使い方では：

* 試行錯誤が増える
* 長い文章を何度も送る

👉 **結果として無料枠をすぐ消費してしまいます**

一方で、

* 補助型 → 必要な部分だけ使う
* 提案型 → まとめて処理する

👉 **無駄な消費を抑えることができます**

---

## 1.3 ステップアップの考え方

慣れてきたら、以下の順でステップアップしてください。

### Step1( 初級 )

* 補助型ツールを使う
* コード生成に慣れる

---

### Step2( 中級 )

* 提案型ツールを使う
* AIに作業を任せる

---

### Step3( 上級 )

* 高性能なモデルに切り替える( 有料 )

#### 理由

* コード品質が向上する
* リファクタリングや設計提案が強くなる

---

## 1.4 注意点( 重要 )

* AIの出力は必ず確認すること
* 分からないコードはそのまま使わないこと
* 機密情報は入力しないこと

👉 **AIは便利なツールですが、最終判断は人間が行います**

---

## 2. AIの構成( エンジンとクライアント )

AIコード作成は、以下の2つの組み合わせで動作します。

* **エンジン( LLM )**：実際にコードを生成する「頭脳」
* **クライアント( VSCode拡張 )**：ユーザーが操作する「ツール」

---

### ■ 基本ルール

* クライアント単体では動作しない
* 必ずエンジンと組み合わせて使用する

👉 **「クライアント × エンジン」の組み合わせが重要です**

## 2.3 代表的な使い方
上記の表( Cxx × Exx )を組み合わせた、代表的な利用パターンを示します。

### ■ パターン①( 初心者・無料 )

* C2 Windsurf( クライアント )
* E4 Windsurf Model( エンジン )

👉 無料でそのまま使える( 一体型 )

---

### ■ パターン②( 中級・柔軟 )

* C5 Continue( クライアント )
* E1 GPT系 / E2 Claude系( エンジン )

👉 エンジンを自由に選べる( 分離型 )

---

### ■ パターン③( 上級・効率重視 )

* C7 Cline( クライアント )
* E2 Claude系( エンジン )

👉 自動実行・提案型で開発効率が高い

---

### ■ 補足( 重要 )

* 一体型：エンジンが内蔵されている( 簡単・無料が多い )
* 分離型：エンジンを自分で選ぶ( 高性能・課金あり )

👉 **初心者は「一体型」から始めるのが安全です**
