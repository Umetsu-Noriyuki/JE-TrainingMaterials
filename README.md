# IT×AI活用ロードマップ

本計画は、最新のITトレンドに基づき、AIを「効率的な学習パートナー」として使いこなしながら、実戦的な開発・運用スキルを身につけることを目的としています。  

## 📋 学習カリキュラム一覧
各STEPの「資料を確認する」から詳細資料を開き、学習を進めてください。
