# Linuxとは何か

LinuxはOS（オペレーティングシステム）の一種です。

OSとは、コンピュータを動作させるための基本ソフトウェアのことを指します。

例えば、皆さんが普段利用しているパソコンにもOSがインストールされています。

代表的なOSには以下があります。

| OS | 主な利用用途 |
|------|------|
| Windows | 一般的なPC |
| macOS | Apple製PC |
| Linux | サーバー、クラウド、組み込み機器など |

私たちが日常的に利用するパソコンでは Windows や macOS が使われることが多く、サーバーでは Linux が利用されることが一般的です。

イメージすると次のようになります。

```text
普段のPC
    ↓
Windows / macOS

サーバー
    ↓
Linux
```

そのため、システム開発やインフラ運用を行うエンジニアは Linux の知識が必要になります。

# Linuxが利用される理由

Linuxが広く利用されている理由はいくつかあります。

## 無償で利用できる

多くのLinuxディストリビューションは無償で利用できます。

そのため、多くの企業や開発者が導入しやすくなっています。

※ 一部のディストリビューションは企業向けサポートを含む有償サービスとして提供されています。

---

## 安定している

Linuxは長期間安定して動作することを重視して設計されています。

そのため、24時間365日稼働するサーバーでも利用されています。

---

## カスタマイズしやすい

必要な機能だけを組み合わせて利用できます。

そのため、

- Webサーバー
- データベースサーバー
- アプリケーションサーバー

など様々な用途に対応できます。

---

## 多くのクラウドサービスで利用されている

現在利用されているクラウドサービスの多くはLinuxを基盤としています。

例えば、

- AWS
- Microsoft Azure
- Google Cloud

などでもLinuxサーバーを利用できます。

そのため、インフラエンジニアだけでなく、アプリケーション開発者もLinuxの知識が求められることがあります。

---

# Windowsとの違い

普段利用しているWindowsとLinuxにはいくつかの違いがあります。

## 操作方法

Windowsではマウスを利用したGUI操作が中心です。

GUI（Graphical User Interface）とは、画面上のボタンやアイコンを利用して操作する方法です。

一方、Linuxではコマンドを入力して操作するCLI（Command Line Interface）がよく利用されます。

例：

```bash
ls
pwd
cd
```

LinuxはCLIが主な操作方法ではありますが、LinuxにもGUI環境は存在します。

しかし、サーバーではCLIが利用されることが多いため、本教材ではCLIを中心に学習します。

---

## ソフトウェアの管理方法

Windowsではインストーラーを実行してソフトウェアを導入することが一般的です。

Linuxではパッケージ管理システムを利用してソフトウェアをインストールします。

例えば、Webサイトを公開するために利用されるWebサーバーソフトウェア「Nginx」をインストールする場合は次のようなコマンドを利用します。

```bash
sudo dnf install nginx
```

詳細は後続の教材で学習します。

---

## 利用される場所

| OS | 主な利用場所 |
|------|------|
| Windows | 個人PC、業務端末 |
| Linux | サーバー、クラウド環境 |

どちらが優れているというわけではなく、用途によって使い分けられています。

# Linuxディストリビューション

Linuxには多くの種類があります。

これらを「ディストリビューション」と呼びます。

## Linuxとディストリビューションの関係

実は、Linuxという名前は本来OS全体を指す言葉ではありません。

LinuxはOSの中心部分である「カーネル」を指します。

カーネルとは、CPUやメモリ、ストレージなどのハードウェアを管理するOSの中核部分です。

しかし、カーネルだけではコンピュータを操作することはできません。

そのため、

- コマンド
- ソフトウェア管理機能
- 設定ツール
- 各種ライブラリ

などを組み合わせて、実際に利用できるOSとしてまとめたものがディストリビューションです。

```text
Linuxカーネル
    +
各種ツール
    +
ソフトウェア管理機能
    ↓
ディストリビューション
```

私たちが普段利用しているUbuntuやRocky Linuxは、このディストリビューションにあたります。

## なぜ複数のディストリビューションが存在するのか

利用目的によって求められる機能が異なるためです。

例えば、

- 学習や開発向け
- 企業サーバー向け
- 組み込み機器向け

など、用途に応じて様々なディストリビューションが作られています。

そのため、Linuxには以下のような多くの種類が存在します。

## Ubuntu

初心者にも利用しやすいディストリビューションです。

特徴

- 情報が豊富
- 導入しやすい
- 学習向け

## Rocky Linux

企業向けサーバーでよく利用されるディストリビューションです。

特徴

- サーバー用途に強い
- 安定性を重視
- 企業利用が多い

## Red Hat Enterprise Linux (RHEL)

企業向けに提供されている商用ディストリビューションです。

特徴

- 商用サポートがある
- 企業利用向け
- 高い信頼性

# まとめ

本章ではLinuxの基本的な概要について学習しました。

理解しておきたいポイントは以下の通りです。

- LinuxはOSの一種である
- サーバーで広く利用されている
- Linuxの中核部分をカーネルと呼ぶ
- 実際に利用するOSはディストリビューションとして提供される

# 用語集

| 用語 | 説明 |
|--------|--------|
| Linux | サーバーやクラウド環境で広く利用されているOSの中核技術 |
| OS | コンピュータを動作させるための基本ソフトウェア |
| カーネル | OSの中心部分。CPUやメモリなどのハードウェアを管理する |
| ディストリビューション | Linuxカーネルと各種ツールを組み合わせた実際に利用できるOS |
| Ubuntu | 学習用途や開発用途で広く利用されるLinuxディストリビューション |
| Rocky Linux | サーバー用途で利用されるLinuxディストリビューション |
| Red Hat Enterprise Linux <br>(RHEL) | 企業向けサポートを提供する商用Linuxディストリビューション |
| GUI | 画面上のボタンやアイコンを利用して操作する仕組み |
| CLI | コマンドを入力して操作する仕組み |
| ターミナル | Linuxでコマンドを入力するための画面 |
| パッケージ管理 | ソフトウェアのインストールや更新を管理する仕組み |
| サーバー | 利用者や他のシステムにサービスを提供するコンピュータやソフトウェア |
