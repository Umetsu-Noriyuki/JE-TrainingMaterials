# AIコード作成サポート(VSCode活用 実践編)

## 1. このステップの目的

このステップでは、実際にVSCodeにAI拡張を導入し、
PythonコードをAI補助で作成できる状態まで進めます。

👉 **「AIがコードを書く体験」をすることが目的です**

---

## 1.1 今回やること

以下の流れで進めます。

1. VSCodeに拡張機能をインストール
2. Python環境を準備
3. AI補助でコードを書く
4. 実行して動かす

👉 **難しいことはやりません。まずは動かすことを優先します**

---

## 1.2 使用する構成( 初学者向け )

今回は「無料・簡単」を優先して以下を使います。

* エディタ：VSCode
* AI拡張：Windsurf Plugin (formerly Codeium)( 無料 )
* 言語：Python

👉 **STEP1-2の「パターンA」に対応した構成です**

---

## 2. VSCodeにAI拡張を入れる

### 2.1 拡張機能の開き方

1. VSCodeを開く
2. 左の「四角いアイコン( 拡張機能 )」をクリック

---

### 2.2 拡張機能の検索( 重要 )

検索欄に以下を入力してください：

```text
Windsurf
```

---

### 2.3 インストールする拡張( 重要 )

以下をインストールしてください：

👉 **Windsurf Plugin (formerly Codeium)**

---

### ⚠ 補足( 必ず読んでください )

* 「Codeium」は現在名称変更されています
* 現在は **Windsurf** という名前になっています
* 「Codeium Enterprise Updater」はインストールしないでください  
  これは企業向けの管理ツールで、AI補助は使えません

---

### 2.4 ログイン

インストール後、ログイン画面が表示されます

* Googleアカウントなどでログイン可能
* 無料で利用できます

---

## 3. Python環境の準備

### 3.1 Pythonが入っているか確認

ターミナルで以下を実行：

```
python --version
```

または

```
python3 --version
```

---

### 3.2 入っていない場合

以下からインストールしてください：

* [https://www.python.org/](https://www.python.org/)

---

### 3.3 VSCodeのPython拡張を入れる

検索欄で：

```
Python
```

→ Microsoft製のものをインストール

---

## 4. 実際にAIでコードを書く

### 4.1 ファイル作成

新しいファイルを作成：

```
test.py
```

---

### 4.2 AI補助を体験する

以下を入力してみてください：
```python
# 数値のリストを作って平均を計算する関数
```
👉 Enterキーを押す or 少し待つと…

👉 **AIがコードを提案してきます**

---

### 4.3 提案を受け入れる
* Tabキー → 採用
* Escキー → 無視

---

### 4.4 例
入力
1. コメントを書いて Enterキー
2. 関数の提案が出て 採用Tabキー
3. 次の行に Enter
2. 上記関数を呼ぶ提案が出て 採用Tabキー

生成コード
Ex.)
```python
# 数値のリストを作って平均を計算する関数
def average(values):
  total = 0
  for x in values:
    total += x
  return total / len(values)

print(average([1, 2, 3, 4, 5]))
```

---

## 5. コードを実行する

ターミナルで実行：

```
python test.py
```

---

### 実行結果

```
20.0
```

👉 **ここまでできれば成功です**

---

## 6. もう一歩やってみる( 重要 )

次は少しだけレベルアップします。

---

### 6.1 指示を少し具体的にする

```python
# ユーザーから入力を受け取って合計を計算するプログラム
```

👉 AIの精度が上がることを確認してください

---

### 6.2 英語コメントでも試す

```python
# create a function to check if a number is prime
```

👉 **英語の方が精度が高いことが多いです**

---

## 7. よくあるつまずき

### ❌ AIが出てこない

* Windsurfが有効か確認
* ログイン済みか確認

---

### ❌ Pythonが動かない

* パスが通っていない
* python / python3 の違い

---

### ❌ コードが間違っている

👉 正常です

* AIは100%正しくありません
* 必ず確認してください

---

## 8. このステップで重要なこと

この段階で覚えてほしいのは以下です：

---

### ✔ AIは「補助」である

* 自動生成ツールではない
* 自分で理解することが重要

---

### ✔ コメントが命

👉 良い指示を書くと、良いコードが出ます

---

### ✔ 小さく試す

* いきなり大きなコードを書かせない
* 小さい単位で確認する
