## 🟦 3. インフラ基礎編( AWSサービス )

### 3-1 インフラ概念( Infrastructure の基本とAWSの土台 )
👉 [資料を確認する](社員のスキルアップ教育計画_0301インフラ基礎編-インフラ概念.md)
* **目的**: インフラの概要を理解し、AWSの代表的な基礎サービスを把握して、効率的なシステム運用の土台を学ぶ。
* **キーワード**: Infrastructure, AWS, EC2, S3, RDS, VPC

### 3-2 管理ツールの活用法( Management Tools )
👉 [資料を確認する](社員のスキルアップ教育計画_0302インフラ基礎編-管理ツール.md)
* **目的**: AWSを安全かつ効率的に操作・管理するための基本ツールを理解し、IAMによる権限設計とセキュリティ強化のポイントを学びます。
* **キーワード**: Management Console, Region, IAM, 権限設計, セキュリティ

### 3-3 ネットワーク・サーバー基盤( VPC、EC2、ECS+EC2、ECS+Fargate、Security Group )
👉 [資料を確認する](社員のスキルアップ教育計画_0303インフラ基礎編-ネットワーク・サーバ.md)
* **目的**: AWS上での安全で効率的なネットワーク構築、通信制御と仮想サーバー・コンテナ基盤運用の基礎を理解し、実務での安全な設計と運用をサポートします。
* **キーワード**: VPC, EC2, ECS, Fargate, Security Group, インフラ基盤, CI/CD, ECR, Docker Hub

### 3-4 サーバーレスの基礎( 管理不要で拡張自在なクラウド実行環境 )
👉 [資料を確認する](社員のスキルアップ教育計画_0304インフラ基礎編-サーバーレス.md)
* **目的**: AWSのサーバーレスサービスの基礎を理解し、管理負荷を減らしながら迅速かつ安全なシステム開発を学ぶ。
* **キーワード**: Lambda, API Gateway, CloudWatch, Serverless, スケーラビリティ

### 3-5 ストレージと高速配信の基礎( S3とCloudFront )
👉 [資料を確認する](社員のスキルアップ教育計画_0305インフラ基礎編-ストレージ.md)
* **目的**: AWSの主要ストレージサービスS3とCDNサービスCloudFrontの概要を理解し、高速かつ安全なデータ管理と配信の基礎を習得する
* **キーワード**: S3, CloudFront, CDN, オブジェクトストレージ, バケット, エッジロケーション

### 3-6 AWSデータベースサービスの基礎理解と選択ポイント( RDS・Athena・DynamoDB )

* **目的**: AWSが提供するRDS、Athena、DynamoDBの特徴と適した利用シーンを理解し、適切なデータベース選択を目指します。
* **キーワード**: RDS, Athena, DynamoDB, NoSQL, SQL, マネージドサービス

### 3-7 定時処理・通知処理( EventBridge と SNS の活用 )
👉 [資料を確認する](社員のスキルアップ教育計画_0307インフラ基礎編-定時通知.md)
* **目的**: AWS EventBridge によるイベント連携と SNS による多彩な通知方法を理解し、業務効率化とリアルタイム情報伝達を目指します。
* **キーワード**: EventBridge, SNS, 定時処理, メッセージ配信, 自動化

### 3-8 イベント処理・連携・分析( AWSで実現する効率的なデータ活用基盤 )

* **目的**: AWSのイベント処理とデータ分析に使う代表的サービスの特徴と実務適用を理解し、効率的な基盤構築を目指します。
* **キーワード**: SQS, EventBridge, Firehose, Glue, Redshift, SNS, ETL, Pub/Sub

### 3-9 クラウド通信( 踏み台サーバ、VPN、Site-to-Site VPN、Direct Connect )
👉 [資料を確認する](社員のスキルアップ教育計画_0309インフラ基礎編-通信.md)

* **キーワード**: 踏み台サーバ, Bastion, VPN, Site-to-Site VPN, Direct Connect, VPC, Security Group, 専用回線, クラウド通信
